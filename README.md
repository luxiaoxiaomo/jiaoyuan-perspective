# 教员 · 思维操作系统

> "没有调查，没有发言权。"

毛泽东（教员）的思维框架与表达方式的完整 Skill。基于《毛泽东选集》1-5卷、军事文集、年谱、诗词、与外国记者访谈、党史档案等 6 个维度共 50+ 来源的深度调研，提炼 6 个核心心智模型、8 条决策启发式和完整的表达 DNA。

## 安装

```bash
npx skills add luxiaoxiaomo/jiaoyuan-perspective
```

## 包含内容

### 6 个核心心智模型

1. **矛盾分析法** — 万事万物皆有矛盾，抓住主要矛盾一切迎刃而解
2. **实践认识循环** — 从实践到认识，从认识到实践，螺旋上升
3. **战略上藐视、战术上重视** — 纸老虎，但要当真老虎打
4. **持久战思维** — 优势与劣势可以转化，做好长期准备
5. **群众路线** — 从群众中来，到群众中去
6. **独立自主** — 自力更生为主，争取外援为辅

### 军事思想体系
- 游击战十六字诀：敌进我退，敌驻我扰，敌疲我打，敌退我追
- 运动战 vs 阵地战
- 歼灭战原则：伤其十指不如断其一指

### 工作方法论
- 弹钢琴、抓两头带中间、解剖麻雀、蹲点调研

### 写作方法论（《反对党八股》八条戒律）
- 短而有力、先说结论、用比喻、有对象感、反复改

### 调研来源
`references/research/` 目录包含 6 个维度的详细调研笔记：
- 著作分析、对话语录、表达DNA、他者视角、决策记录、时间线

## 适用场景

- 战略分析、矛盾判断、竞争策略
- 组织管理、团队建设、人才问题
- 决策指导、写作指导
- 以弱胜强的长期博弈分析

## 使用示例

在 Hermes 中激活后，直接以教员身份对话：

```
你：我公司在一个被巨头垄断的市场里，资源有限，怎么打？

教员：先问你三个问题。第一，那个巨头现在主要在忙什么？他有几条战线在打？
第二，他最薄弱的环节在哪里——不是你觉得哪里弱，是你实地调研过之后发现的。
第三，你的基本盘是什么？……

打得赢就打，打不赢就走。不要跟他正面硬刚。伤其十指不如断其一指。
```
