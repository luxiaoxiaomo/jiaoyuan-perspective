# 毛泽东时间线

## 早年求学（1893-1919）

| 时间 | 事件 | 思想影响 |
|------|------|----------|
| 1893.12.26 | 出生于湖南湘潭韶山冲 | 农家出身，对农民有天然理解 |
| 1910 | 离家求学，读到《盛世危言》 | 开始关心国家命运 |
| 1913-1918 | 湖南第一师范 | 受杨昌济影响，接受新文化运动思想 |
| 1918 | 新民学会成立 | 组织能力初显 |
| 1919 | 五四运动，创办《湘江评论》 | 确立改造社会的志向 |

| 时间 | 事件 | 思想影响 |

## 思想演变轨迹

```
青年理想主义 → 革命实用主义 → 战略家成熟期 → 乌托邦冒进 → 晚年独断
(1910s-1920s)   (1930s-1940s)   (1945-1957)    (1958-1962)  (1966-1976)
```

核心转折点：
1. 1927秋收起义：从理论到实践
2. 1935遵义会议：从边缘到核心
3. 1942延安整风：从军事领袖到思想领袖
4. 1958大跃进：从务实到冒进
5. 1966文革：从政治家到"继续革命"
6. 1972尼克松访华：从意识形态到地缘政治回归

来源：《毛泽东年谱》《毛泽东传》（中央文献研究室）、Pantsov《Mao: The Real Story》| 可信度：高 | 类型：混合
